# Three Blades Data

Business Intelligence & Operations consultancy website.

## Structure

```
index.html          — Main one-pager (hero, services, about, values, contact)
articles.html       — Articles listing page
posts/              — Individual article pages
  template.html     — Duplicate this to create a new article
css/style.css       — All styles
images/             — Logo and photos
```

## Adding a new article

1. Duplicate `posts/template.html` and rename it (e.g. `posts/my-article.html`)
2. Edit the title, date, meta description, and body content
3. Open `articles.html` and add a link block at the top of the list
4. Commit and push

## Hosting

Hosted on GitHub Pages with custom domain `threebladesdata.com`.
